# Icons

Place these files here:

- `icon-192.png` — 192×192 app icon (used for Android PWA and iOS home screen)
- `icon-512.png` — 512×512 app icon (used for splash screens and high-DPI)

Both should have a transparent or dark (`#03000b`) background to match the RaBbLE void aesthetic.
Recommended: the RaBbLE particle/entity logo on the dark background.
